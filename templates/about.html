What is HCTCG?

HCTCG stands for Hermitcraft Trading Card Game. It was created by VintageBeef, a member of the hermitcraft server.
The original game was made within minecraft, using map art as the cards, After the game grew in popularity with both
the players and viewers alike, the game was adapted into a physical TCG (Trading Card Game), with Beef working
alongside CREO, a company that manufactures card games.

What is this Website?

The purpose of this website is to provide people a place to look up the different cards, their stats, and how they are ruled.
This website as of writing uses only the map art images of cards, but may include the cards from the physical version
in the future.
This website was made for a school programming project in which we need to create a website using Flask, a WSGI
web application framework for Python. Backed with an SQLite Database for storing the data, along with Jinja for
templating HTML pages.
If you are reading this (and you aren't my teacher!) then it most likely means I've gotten the website into a state where I
am able to release it for people to use, a goal that I wasn't really intending, but would be nice to do.

How/Where can I play HCTCG?

Buy The Cards! - There is a huge community out there dedicated to HCTCG, you can find people to play a game via webcam,
Or even organize to play in person via the CREO Discord. (Be safe out there!)

HCTCG Online! - A fan made/run website in which you can play the game online using the cards from minecraft! It
includes deck building, public matchmaking, and even boss fights!

Wildcard MC! - A minecraft survival server with a thriving, friendly community, with one of it's main focuses being
to be a place where fans of HCTCG and hermitcraft can hang out and play the game!

Garibaldi's Decked out 2 Server! - A server decicated purely to playing both HCTCG and Decked out 2 by TangoTek,
this server is my personal home ground, where I and a group of others take care of keeping the game running!
